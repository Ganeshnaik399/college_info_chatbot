# 🎓 College Info Chatbot  

A modern, interactive chatbot designed to provide information about a college — its courses, timings, faculty, placements, internships, and much more.  
Built with **HTML, CSS, and JavaScript**, this chatbot gives a user-friendly interface for students to quickly get answers about the college.  

---

## 🚀 Features  

- ✅ Interactive two-panel design (Menu + Chat window)  
- ✅ Quick buttons for common questions (Timings, Courses, Placements, etc.)  
- ✅ Animated typing indicator before showing bot responses  
- ✅ Smooth fade-in animations for messages  
- ✅ Custom scrollbar for clean design  
- ✅ Responsive layout (works on mobile)  
- ✅ Emojis added for better UX  
- ✅ Well-organized code structure (HTML, CSS, JS)  

---

## 🛠️ Technologies Used  

- ✅ HTML5  
- ✅ CSS3 (Flexbox, Animations, Custom Scrollbar)  
- ✅ JavaScript (DOM manipulation)  
- ✅ Boxicons (for icons)
